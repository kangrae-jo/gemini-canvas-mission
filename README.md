# gemini-canvas-mission

Gemini Canvas 미션 결과물을 관리하는 저장소입니다.


## 목적

- Gemini Canvas로 제작한 웹앱 결과물을 기록하고 공유합니다.
- 앱별 요구사항, 배포 링크, 회고를 일관된 형식으로 남깁니다.


## 💻 1. 유틸리티 앱

### 앱 이름
> 조작된 룰렛

### 배포 링크
> [조작된 룰렛](https://gemini.google.com/share/43671c6e41f3)

### 주요 기능

- 선택지를 등록하는 기능
- 여러 선택지 중 하나만 선택하는 랜덤 당첨 기능
- 각 선택지에게 가중치를 부여하고 당첨 확률을 다르게 조작하는 기능

## 🤝 4. 페어 프롬프트 릴레이 앱

### 앱 이름
> 툭 (took)

### 페어
@dahxxn

### 배포 링크
> [툭(took)](https://gemini.google.com/share/3fa42f5e7b09)

### 주요 기능

- 자연어 텍스트 기반 타임라인 생성: 일상적인 문장으로 일정을 '툭' 입력하면, 시간 순서대로 자동 정렬된 깔끔한 타임라인 UI로 즉시 변환되어 나타납니다.
- 스케줄 누적 및 자동 재정렬: 일정을 생각날 때마다 추가로 입력해도, 기존 타임라인 데이터와 합쳐져 올바른 시간대(오름차순) 위치에 자동으로 끼워 맞춰집니다.
- 스마트 유효성 검사 (Error Handling): "25시", "99분" 같은 현실에 존재하지 않는 잘못된 시간을 입력하면, 앱이 이를 감지하여 어떤 시간이 왜 잘못되었는지 구체적인 사유와 함께 사용자에게 수정을 요청합니다.
- 지난 일정 자동 비활성화 (Time Tracking): 현재 시간(브라우저 기준)을 실시간으로 추적하여, 이미 시간이 지나버린 일정은 타임라인에서 회색으로 흐려지고 취소선이 그어져 남은 일정에 집중할 수 있게 돕습니다.
